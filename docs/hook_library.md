## How to Use This Library

Each hook is structured as:

```
### Hook Template
**Category:** [Emotion Category]
**Format:** [List | Story | Question | Revelation | Stat | Challenge]
**Example:** "[Use this exact format — fill in the blank]"
**Visual Style:** [Background, text treatment, animation for video overlay]
```

### Pipeline Integration
The video_engine should:
1. Pick 1 hook from the library per video (rotated)
2. Use the **visual style** to generate/select background footage and text animations
3. Insert the hook as a **2-3 second text overlay** at the start of the video
4. Append the CTA as the **final frame** (5 second outro)

### Hook-to-Emotion Legend

| Emotion Code | Emotion | Best For | Engagement Signal |
|-------------|---------|----------|-------------------|
| 🔍 CUR | Curiosity | All niches — opens a knowledge gap | Watch time |
